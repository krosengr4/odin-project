# AXES

- Flexbox can work either horizontally or vertically and some rules change depending on which direction you work in.
- The default direction for a flex container is horizontal or row. This can be manually changed to vertical or column.

- Think of flex-containers as having 2 axes: the main axis and the cross axis
- In most circumstances:
    - "flex-direction: row;" puts the main axis horizontal
    - "flex-direction: column;" puts the main axis vertical
