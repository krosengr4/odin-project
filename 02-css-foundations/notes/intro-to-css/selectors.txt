*** INTRO TO CSS ***

- Basic Syntax
    - At the most basic level, CSS is made up of various rules.
    - Each rule is made up of a selector and a semicolon-seperated list of declarations.
        - Each of these declarations are made up of a property-value pair.
    - Ex:
        - div.bold-text {
            font-weight: 700;
        }

- Selectors
    - Selectors refer to the HTML elements to which CSS rules apply.
    - They are what is actually being "selected" for each rule.

- The Universal Selector
    - Will select elements of every type(the whole document)
    - This is usually determined with a *
    - Ex(every element has color purple applied):
        - * {
            color: purple;
        }

- Type Selectors
    - Will select all elements of a given type
    - Ex(All <p>'s are colored red):
        - p {
            color: red;
        }

- Class Selectors
    - Will select all elements of a given class
    - Classes are attributes that are given to an HTML element
    - These are signified in CSS by starting it with a "."
    - Ex(all "alert-text" class are yellow):
        - .alert-text {
            color: red;
        }

- ID Selectors
    - Similar to class selectors, but select an element with the given ID
    - Instead of using a "." we use a "#" for ID selectors.
    - Ex(all "title" ID's have a background color of red):
        - #title {
            background-color: red;
        }
    - A common pitfall is people overusing the ID attribute when they don't necessarily need to.

- The Grouping Selector
    - We can group selectors together using a comma seperated list.
    - Ex(both classes have white background and black color):
        - .read,
          .unread {
              color: black;
              background-color: white;
          }

- Chaining Selectors
    - If you have two elements with the subsection class that you want to apply seperate rules for, you can chain them
    - This works for both classes(.) and ID's(#)
    - Ex(html):
        - <div>
            <div class="subsection header">Foo</div>
            <p class="subsection preview">Bar</p>
        <div>
    -Ex(CSS):
        .subsection.header {
            color: red;
        }
        .subsection.preview {
            color: blue;
        }

- Descendant Combinator
    - Represented in CSS by a single space between selectors.
    - Will only cause elements that match the last selector to be selected if they also have an ancestor that matches the previous selector.
    - Ex: .ancestor .child
    - This will select child elements no matter how deep the nesting is.
